AI-gebruik
Voor deze opdracht heb ik ChatGPT gebruikt als extra hulpmiddel tijdens het maken van mijn project.

Hoe ChatGPT mij geholpen heeft

- Ik heb ChatGPT gebruikt om extra uitleg te krijgen over stukken leerstof die we in de les hebben gezien.
- Ik heb gevraagd of mijn idee haalbaar was met de leerstof die we tot nu toe behandeld hebben.
- Ik heb een testversie laten tonen om beter te begrijpen hoe ik aan de opdracht kon beginnen en hoe zoiets ongeveer werkt.
- Ik heb hulp gevraagd bij de betekenis van bepaalde stukken JavaScript-code.
- Ik heb ook gevraagd hoe ik bepaalde code of functies zelf kon gebruiken binnen mijn opdracht.
- Daarnaast heb ik ChatGPT gebruikt om fouten in mijn code beter te begrijpen en om te helpen bij debugging.

Belangrijk
Ik heb ChatGPT vooral gebruikt voor uitleg, ondersteuning en debugging. De code moest ik wel zelf begrijpen, aanpassen en toepassen binnen mijn eigen opdracht.
